## Experiment 5: Comparative Study of Prompting Techniques

## Aim
To test and compare how different prompting techniques such as Basic Prompting, Zero-Shot Prompting, Few-Shot Prompting, and Chain-of-Thought Prompting affect the quality, accuracy, and depth of responses generated by an AI model.

## AI Tool Used
ChatGPT (OpenAI Language Model)

## Purpose: Generate responses based on different prompting strategies.
Evaluation: Analyze how prompt structure influences AI reasoning and output quality.

## 4. Chain-of-Thought Prompt
The prompt explicitly asks the model to reason step-by-step before arriving at the final answer.

Prompt: Explain how artificial intelligence is used in everyday life.

Think step by step:

Identify common daily activities where technology is used.
Determine where AI helps automate or improve these activities.
Provide examples such as smartphones, navigation apps, and online shopping.
Explain how AI improves each activity.
Provide the explanation clearly.

## 📊 Analysis

Prompting Technique	Prompt Characteristics	Response Quality	Observations
Basic Prompt	Simple instruction with minimal guidance	General and brief response	Broad explanation with limited structure and fewer examples. Lack of detail resulted in less informative output.
Zero-Shot Prompt	Clear instructions but no examples provided	More structured and informative	Produced clearer explanations and included multiple real-world examples. Specific instructions improved organization.
Few-Shot Prompt	Includes sample examples before the task	Well-structured and consistent	AI followed the format of the given examples exactly. Examples helped guide the structure and content consistency.
Chain-of-Thought	Asks the model to reason step-by-step	Detailed and logically organized	AI explained the topic systematically. This method improved depth, clarity, and logical reasoning significantly.
Summary of Analysis
The experiment shows that prompt structure strongly influences the quality of AI-generated responses. Basic prompts generate general answers, while structured prompts improve clarity and reasoning. Among all techniques, Few-Shot and Chain-of-Thought prompting produced the most organized and informative responses.

## 🏁 Conclusion

From this experiment, it is concluded that prompt engineering techniques significantly influence the performance of AI models. Clear and structured prompts help the AI generate more meaningful and organized responses. Among the techniques tested, few-shot prompting and chain-of-thought prompting provided better explanations and improved response quality compared to basic and zero-shot prompts. Selecting the appropriate prompting technique is critical for obtaining accurate and informative results from AI systems.

## ✅ Result

The experiment was conducted by applying different prompting techniques (Basic, Zero-Shot, Few-Shot, and Chain-of-Thought) using an AI language model. The basic prompt produced a general response, while the zero-shot prompt improved clarity through specific instructions. The few-shot prompt generated a structured answer by using format guidance. Finally, the chain-of-thought prompt produced the most detailed and logically organized response by encouraging step-by-step reasoning.
